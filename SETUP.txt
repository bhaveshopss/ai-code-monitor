============================================
  AI CODE MONITOR — SETUP COMMANDS
============================================

STEP 1: Start the monitor
-----------------------------------------
npx ai-code-monitor


STEP 2: Configure Claude Code
-----------------------------------------

export CLAUDE_CODE_ENABLE_TELEMETRY=1
export OTEL_METRICS_EXPORTER=otlp
export OTEL_LOGS_EXPORTER=otlp
export OTEL_EXPORTER_OTLP_PROTOCOL=http/json
export OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4318
claude


STEP 3: Open dashboard
-----------------------------------------
http://localhost:3000


============================================
  NOTES
============================================

- Codex (npm @openai/codex): Does NOT support OTel export yet.
  The Rust binary (codex-rs) has OTel support but the npm version does not.

- OpenCode / Crush: Does NOT export OTel data.

- Any tool that exports OTLP over HTTP will work.
  Just set OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4318


============================================
  CLEANUP
============================================

Unset Claude Code env vars:
unset CLAUDE_CODE_ENABLE_TELEMETRY OTEL_METRICS_EXPORTER OTEL_LOGS_EXPORTER OTEL_EXPORTER_OTLP_PROTOCOL OTEL_EXPORTER_OTLP_ENDPOINT
